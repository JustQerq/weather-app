# Overview

Weather app frontend by Oleg Samarets, part of a test assignment for Project Manager Accelerator.
- Written in ReactJS
- Uses weather, geolocation and LLM APIs
- Interacts with the backend (https://github.com/JustQerq/Weather_app_backend/tree/main) through a RESTful API

# How to run
- Clone this repository
- Run `npm install` in the project's folder
- Run the backend application: https://github.com/JustQerq/Weather_app_backend/tree/main
- Use `npm run dev` in the project's folder
- Open `http://localhost:3000/` in browser
